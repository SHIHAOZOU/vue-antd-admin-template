## 介绍📖
vue-antd-admin-template（uu Admin）是一个极简的后台管理系统解决方案，它基于 Vue3 和 Ant Design Vue 实现。它使用了最新的前端技术栈 Vue3 Setup语法糖 & Vite3 & Vue-Router4 & Ant Design Vue & Pinia & TypeScript 。

目前正在持续更新中，对于项目有不完善的地方，希望大家能多提意见。（点Star⭐🤣）
